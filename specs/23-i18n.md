## Context

Currently content in the game is hardcoded in Spanish:
- Event titles ("La Huelga del Silencio", "El Laboratorio Abierto")
- Action messages
- NPC names
- Narrative text in manifestos
- UI labels and messages

For broader reach and testing, we need language support with Spanish as default and English as option.

## Proposal

### Environment Variable

```bash
ECO_LANG=es  # Spanish (default)
ECO_LANG=en  # English
```

### Scope: Content That Needs Translation

| Category | Examples | Priority |
|----------|----------|----------|
| Event titles/summaries | "La Huelga del Silencio" | high |
| Action messages | "Wrote manifesto", "Found circle" | high |
| Fallback event pool | 6-8 hardcoded events | high |
| NPC names | "Dra. Maela Ruun" | medium |
| Manifestos | AI-generated (prompt language) | medium |
| Error messages | "Action failed", "Invalid target" | low |
| UI labels | Status messages, help text | low |

### Content NOT Requiring Translation

- Essence names: always English identifiers (anarchism, technocracy, etc.)
- Tag names: system identifiers, not display text
- File paths and technical identifiers

### Loader (`__init__.py`)

```python
import os
import yaml
from pathlib import Path

LANGUAGE = os.getenv("ECO_LANG", "es")
SUPPORTED = ["es", "en"]

if LANGUAGE not in SUPPORTED:
    LANGUAGE = "es"  # fallback to Spanish

# Load YAML file
I18N_DIR = Path(__file__).parent
_strings = yaml.safe_load((I18N_DIR / f"{LANGUAGE}.yaml").read_text())

def t(key: str, **kwargs) -> str:
    """Get translation. Key format: 'section:subsection:key'"""
    keys = key.split(":")
    value = _strings

    for k in keys:
        if isinstance(value, dict) and k in value:
            value = value[k]
        else:
            return key  # Return key if not found

    if kwargs:
        return value.format(**kwargs)
    return value
```

### Key Format Convention

Keys use `: ` as separator:
- `events:la_huelga_del_silencio:title`
- `actions:write_manifesto:success`
- `messages:no_targets`

### Fallback Behavior

- If YAML file missing: raise error (fail fast)
- If key not found in YAML: return key itself (e.g., `events:missing:title`)
- If LANGUAGE invalid: default to Spanish

### LLM Content Language

When generating AI content, pass the language:
```python
LANG_PROMPT = {"es": "en español", "en": "in English"}[LANGUAGE]
# Include in AI prompt: f"Write the manifesto {LANG_PROMPT}"
```
