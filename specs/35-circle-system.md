## 1. Qué es un Círculo

Un Círculo es una entidad social que agrupa Echos y NPCs alrededor de una esencia compartida. Es más que un contador — tiene identidad, historia, y comportamiento.

```
CIRCLE ENTITY:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

class Circle(BaseModel):
    id: str
    name: str                    # "Circle of the Burning Garden"
    essence: str                 # Esencia dominante
    members: list[str] = []      # Echo IDs activos
    founding_tick: int
    influence: float = 10.0    # Capacidad de atraer
    history: list[CircleEvent]   # Log de eventos
    status: CircleStatus = ACTIVE  # ACTIVE | DORMANT | DISSOLVED
```

## 2. Para qué sirven los Círculos

```
FUNCIONES:
━━━━━━━━━━━

1. NPC SPAWNING (spec-28)
   → Círculo con 3+ miembros tiene 20% chance de spawn NPC
   → NPC hereda esencia del círculo

2. MODIFICADOR DE ACCIONES
   → talk: +bonus si el target está en tu círculo
   → recruit: más fácil en círculos con alta afinidad
   → ritualize: más efectivo en círculos grandes

3. AFINIDAD CON ESENCIAS
   → Círculos atraen Echos de esencia compatible
   → Repelen Echos de esencia opuesta
   → Afinidad = cosine_similarity(circle.essence, echo.essence)

4. REPRESENTACIÓN DE ORGANIZACIÓN
   → Más círculos = más presión sobre autoridad
   → Círculo crece = influencia del grupo aumenta
   → Círculo se disuelve = impacto negativo en mundo

5. TRACKING DE GRUPO SOCIAL
   → Miembros pueden pertenecer a múltiples círculos
   → Unirse/leave tiene costo social
   → Círculo con 6+ miembros puede "splinter" (dividirse)
```

### 3.3 Unicidad

```python
def generate_unique_circle_name(essence: str, existing_circles: list[Circle]) -> str:
    """Generate name that's unique among existing circles."""
    
    names_pool = ESSENCE_NAME_TEMPLATES.get(essence, [
        f"Circle of the {adj} {noun}"
        for adj in ADJECTIVES_POOL
        for noun in NOUNS_POOL
    ])
    
    used_names = {c.name for c in existing_circles}
    
    for name in names_pool:
        if name not in used_names:
            return name
    
    # Fallback: add number
    for i in range(1, 100):
        name = f"{names_pool[0]} {i}"
        if name not in used_names:
            return name

def found_circle(...) -> ActionResult:
    templates = load_circle_templates()

## 4. Sistema de Miembros

### 4.1 Unirse a un Círculo

```
JOIN_CIRCLE ACTION:
━━━━━━━━━━━━━━━━━━━

Requirements:
  - Echo must be in same location as circle (or anywhere, simplified)
  - Circle status != DISSOLVED
  - Echo not already member

Effects:
  - echo.circles.append(circle.id)
  - circle.members.append(echo.id)
  - circle.influence += 2
  - circle.history.append(CircleEvent(type=JOIN, echo_id=echo.id, turn=world_tick))
```

### 4.2 Dejar un Círculo

```
LEAVE_CIRCLE ACTION:
━━━━━━━━━━━━━━━━━━━━

Effects:
  - echo.circles.remove(circle.id)
  - circle.members.remove(echo.id)
  - circle.influence -= 3
  - circle.history.append(CircleEvent(type=LEAVE, echo_id=echo.id, turn=world_tick))
  
  - If circle.members.length == 0 → circle.status = DISSOLVED
```

### 4.3 Crecimiento Orgánico

```
CIRCLE GROWTH (per world tick):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Conditions for growth:
  - circle.influence > 15
  - circle.member_count < 8
  - circle.status == ACTIVE

Chance: 15% per world tick

→ Attract random Echo with affinity > 0.3
→ Or spawn NPC if member_count >= 3

CIRCLE DECAY (per world tick):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Conditions for decay:
  - circle.member_count > 1
  - random.random() < 0.03 (3% chance)

→ Member with lowest resonance leaves
→ circle.influence -= 2
→ circle.history.append(CircleEvent(type=DECAY, turn=world_tick))
```

## 6. Crecimiento y Declive

### 6.1 Círculo crece a 6+ miembros

```
SPLINTER EVENT:
━━━━━━━━━━━━━━━━

When: circle.member_count >= 6 and random.random() < 0.1

→ Circle splits into two
→ Original circle keeps 3 members (random which)
→ New circle has 3 members + new name
→ Both circles have slightly different essences (drift)
→ circle.history.append(CircleEvent(type=SPLINTER, turn=world_tick))

Output:
[Turn 15] ⭕ Circle of the Burning Garden splintered
         → New circle: Circle of the Echoing Voice (3 members)
```

### 6.2 Círculo se disuelve

```
DISSOLUTION:
━━━━━━━━━━━━

When: circle.member_count == 0

→ circle.status = DISSOLVED
→ circle.history.append(CircleEvent(type=DISSOLVED, turn=world_tick))
→ world.circles.remove(circle.id) or mark as dissolved

Output:
[Turn 20] ⭕ Circle of the Burning Garden dissolved
         (last member left)
```

### 6.3 Círculo dormante

```
DORMANT STATUS:
━━━━━━━━━━━━━━━━

When: circle.member_count == 0 but recent activity (last 5 turns)

→ circle.status = DORMANT (not yet dissolved)
→ Can be revived if new member joins

After 10 turns with 0 members → DISSOLVED
```

## 7. Afinidad con Esencias

```python
def get_circle_echo_affinity(circle: Circle, echo: Echo) -> float:
    """Returns affinity between circle and echo (0.0 to 1.0)."""
    
    circle_essence_vector = essence_to_vector(circle.essence)
    echo_essence_vector = essence_to_vector(echo.essence)
    
    return cosine_similarity(circle_essence_vector, echo_essence_vector)
```

```
AFFINITY EFFECTS:
━━━━━━━━━━━━━━━━━

affinity > 0.7:  # High compatibility
  → +20% chance echo joins
  → +10% effectiveness on circle actions
  → Echo gains resonance when in circle

0.4 < affinity < 0.7:  # Neutral
  → Normal behavior

affinity < 0.3:  # Low compatibility
  → -20% chance echo joins
  → -10% effectiveness on circle actions
  → Echo may leave if other options exist
```

## 8. Círculo en el World Tick

```python
def circle_world_tick(circle: Circle, world: World) -> None:
    """Called each world tick for each active circle."""
    
    # Growth check
    if circle.influence > 15 and circle.member_count < 8:
        if random.random() < 0.15:
            attract_echo_to_circle(circle, world)
    
    # Decay check
    if circle.member_count > 1:
        if random.random() < 0.03:
            remove_lowest_resonance_member(circle, world)
    
    # Dissolution check
    if circle.member_count == 0:
        if circle.dormant_turns >= 10:
            circle.status = CircleStatus.DISSOLVED
        else:
            circle.dormant_turns += 1
    
    # Splinter check
    if circle.member_count >= 6:
        if random.random() < 0.1:
            splinter_circle(circle, world)
```

## 9. NPC Spawning desde Círculo

```python
def maybe_spawn_npc(circle: Circle, world: World) -> NPC | None:
    """Check if circle should spawn NPC."""
    
    if circle.member_count < 3:
        return None
    
    if random.random() > 0.2:
        return None
    
    # Spawn NPC
    npc_type = pick_npc_type_for_essence(circle.essence)
    npc = NPC.from_template(npc_type, circle.essence)
    
    circle.members.append(npc.id)
    circle.history.append(CircleEvent(
        type=NPC_SPAWN,
        npc_id=npc.id,
        turn=world.clock.action_tick
    ))
    
    return npc
```

## 10. Acciones Involucrando Círculos

```
FOUND_CIRCLE:
  → Creates new circle with echo as first member
  → circle.influence = 15 (founder bonus)
  → world.pressure += 1
  → world.legitimacy -= 2

JOIN_CIRCLE:
  → Requires target circle ID
  → +2 influence to circle
  → echo joins

LEAVE_CIRCLE:
  → echo leaves circle
  → circle loses member, influence -= 3

TALK (in circle):
  → Bonus to NPC affinity if in same circle
  → Circle influence affects dialogue outcomes

RITUALIZE (in circle):
  → More effective if echo is in circle
  → Circle members can participate (spread effect)
  → +resonance to all members
```
