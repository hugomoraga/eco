## Principio Fundamental

**Separación estricta de concerns (Arquitectura Hexagonal):**

| Capa | Responsabilidad | Importa |
|------|----------------|---------|
| `core/` | Lógica pura, simulación, dominio | Solo stdlib + datos externos |
| `infra/` | Adaptadores de infraestructura (AI, config, logging) | `core` (implementa ports) |
| `adapters/` | Driving adapters (input, UI, TUI) | `core` + `infra` |
| `ui_core/` | Display, colores, widgets, historial | `core` (leer estado) |

**Regla:** `core` NO puede importar `infra`, `adapters`, ni `ui_core`.

class Console:
    _instance: "Console | None" = None

    @classmethod
    def get(cls) -> "Console":
        if cls._instance is None:

        return cls._instance

    def print(self, *args, **kwargs)

## 3. Flujo de un Turno

```
SimulationEngine.run():
  1. turn += 1
  2. world.clock.advance(1)
  3. world.evolve_metrics(rng)  # drift en pressure, legitimacy, resources_global
  4. _notify("on_turn_start", turn, world)
     → ConsoleDisplay.on_turn_start()
        → turn_header + world_metrics_table (TODAS las métricas)

  5. input_source.get_action(turn, world)
     → AutoplayInputSource: devuelve None
     → PlayerInputSource: muestra menú, espera input (timeout)
     → HybridInputSource: cada N turns humano, resto None

  6. Si action_name is None AND autoplay:
       autoplayer_engine.select_action(echo, world, actions)

  7. _notify("on_action_selected", turn, action_name)

  8. action.execute(echo, world, context)
     try:
       result = action.execute(...)
       _notify("on_action_result", turn, action_name, result)
       → ConsoleDisplay.on_action_result()
          → action_result + action_detail
     except Exception as e:
       _notify("on_error", ...)  # no crash

  9. events.evaluate(world)
  10. _notify("on_event", turn, event.type, event.title, event.summary)
      → ConsoleDisplay.on_event() → event_banner

  11. _notify("on_turn_end", turn, world, action_name)
      → ConsoleDisplay.on_turn_end()
         → status_line + metrics_delta_table (si changed)
```

## 7. Métricas de World — Display Completo

### Extraídas de World.stability / World.pressure / World.legitimacy

| Métrica | Fuente en World | Descripción |
|---------|-----------------|-------------|
| Stability | `world.stability` (float 0-100) | Estabilidad general de la civilización |
| Civil Unrest | `world.pressure` (float 0-100) | Tensión social / inestabilidad |
| Authority | `world.legitimacy` (float 0-100) | Legitimidad del liderazgo |
| Resources (G) | `world.resources_global` (float 0-100) | Recursos globales agregados |

### Extraídas de World.resources (dict)

| Métrica | Fuente en World | Descripción |
|---------|-----------------|-------------|
| Food | `world.resources["food"]` | Disponibilidad de alimento |
| Infrastructure | `world.resources["infrastructure"]` | Nivel de infraestructura |
| Energy | `world.resources["energy"]` | Energía disponible |
| Knowledge | `world.resources["knowledge"]` | Conocimiento acumulado |
| Legitimacy | `world.resources["legitimacy"]` | Legitimidad (recurso) |

### Population

| Métrica | Fuente | Descripción |
|---------|--------|-------------|
| Population | `world.population` (int) | Población total |

### Entity Counts

| Métrica | Fuente | Descripción |
|---------|--------|-------------|
| Echoes | `len(world.echoes)` | Ecos activos |
| Circles | `len(world.circles)` | Círculos formados |
| Factions | `len(world.factions)` | Facciones políticas |
| Persons | `len(world.persons)` | Personas (NPC + player) |
| Hosts | `len(world.hosts)` | Hosts activos |

### Influence (Computed, Not Stored)

| Métrica | Fuente | Descripción |
|---------|--------|-------------|
| Influence (F+C) | `sum(f.influence for f in world.factions) + sum(c.influence for c in world.circles)` | Influencia total de facciones + círculos |
